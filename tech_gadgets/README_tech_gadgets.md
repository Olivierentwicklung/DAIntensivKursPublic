# 🚀 Tech gadgets App

Displays a list of gadgets and their manufacturers using **Django Templates** and a **dummy-data JSON**.

## ⚙️ Features

- Displays a list of gadgets
- Displays a list of manufacturers

## 🧪 Example Usage

- Get the app in Browser

  ```bash
   In Browser: http://127.0.0.1:8000/tech_gadgets/
  ```

---

## 🧠 What I Learned

- How to filter items by category in `views.py` using a function
- How to render data in a Django template

- How to use a for loop in a Django template
